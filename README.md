Implementações do Subleq em C, JS e processador em Verilog para a disciplina de Arquitetura de Computadores na Universidade Federal de Sergipe (UFS)


## Como executar

### Em C

Para executar em C, utilize 
```sh
./C/subleq ./Exemplos/[Exemplo]
```
O programa em C mostra passos da execução, bem como de impressão, para depuração do código.

### No navegador

Para executar no JS, abra o HTML, e copie os valores em 'Exemplos' com exceção da primeira linha.

A versão do navegador permite a execução passo a passo do programa, além de exibir todos os endereços de memória e alterações a cada passo executado.
